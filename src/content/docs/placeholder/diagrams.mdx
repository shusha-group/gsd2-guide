This page demonstrates Mermaid diagram rendering via the `@pasqal-io/starlight-client-mermaid` plugin. Each diagram type validates that the plugin correctly processes fenced code blocks into SVG output.

## Flowchart: GSD Auto-Mode Workflow

```mermaid
flowchart TD
    A[Milestone Defined] --> B[Slice Decomposition]
    B --> C{Slices Independent?}
    C -->|Yes| D[Parallel Execution]
    C -->|No| E[Sequential Execution]
    D --> F[Scout Agent]
    E --> F
    F --> G[Planner Agent]
    G --> H[Task Plan Created]
    H --> I[Worker Agent]
    I --> J{Task Verified?}
    J -->|Pass| K[Mark Task Done]
    J -->|Fail| L[Debug & Retry]
    L --> I
    K --> M{More Tasks?}
    M -->|Yes| I
    M -->|No| N[Slice Complete]
    N --> O{More Slices?}
    O -->|Yes| F
    O -->|No| P[Milestone Complete]
```

## Sequence Diagram: Agent Interaction

```mermaid
sequenceDiagram
    participant User
    participant Orchestrator
    participant Scout
    participant Planner
    participant Worker
    participant Reviewer

    User->>Orchestrator: Define milestone
    Orchestrator->>Scout: Explore codebase
    Scout-->>Orchestrator: Context document
    Orchestrator->>Planner: Decompose work
    Planner-->>Orchestrator: Task plan
    loop For each task
        Orchestrator->>Worker: Execute task
        Worker->>Worker: Build & test
        Worker-->>Orchestrator: Task summary
    end
    Orchestrator->>Reviewer: Review slice
    Reviewer-->>Orchestrator: Approval / feedback
    Orchestrator-->>User: Slice complete
```

## State Diagram: Slice Lifecycle

```mermaid
stateDiagram-v2
    [*] --> Planned: Slice plan created
    Planned --> Scouting: Scout assigned
    Scouting --> Planning: Context gathered
    Planning --> Executing: Tasks decomposed
    Executing --> Executing: Next task
    Executing --> Blocked: Blocker discovered
    Blocked --> Replanning: Plan invalidated
    Replanning --> Executing: New plan ready
    Executing --> Verifying: All tasks done
    Verifying --> Complete: All checks pass
    Verifying --> Executing: Checks failed
    Complete --> [*]
```
