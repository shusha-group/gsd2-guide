## Syntax Highlighting

### JavaScript

```js title="scripts/prebuild.mjs"
import { readdir, readFile, writeFile, mkdir } from 'node:fs/promises';
import { join, dirname } from 'node:path';

const SOURCE_DIR = 'content/generated/docs';
const TARGET_DIR = 'src/content/docs';

async function processFile(relativePath) {
  const content = await readFile(join(SOURCE_DIR, relativePath), 'utf-8');
  const lines = content.split('\n');

  // Extract title from first heading
  const titleLine = lines.find(l => l.startsWith('# '));
  const title = titleLine?.replace(/^#\s+/, '') ?? 'Untitled';

  return `---\ntitle: "${title}"\n---\n\n${content}`;
}
```

## Line Highlighting

Use line markers to draw attention to specific lines:

```js {4-6} title="astro.config.mjs"
import { defineConfig } from 'astro/config';
import starlight from '@astrojs/starlight';

// These three lines configure the site for GitHub Pages
const SITE = 'https://gsd-build.github.io';
const BASE = '/gsd2-guide';

export default defineConfig({
  site: SITE,
  base: BASE,
  integrations: [starlight({ title: 'GSD 2' })],
});
```

## Frame Types

### Code Frame (default)

```ts title="src/utils/helpers.ts"
export function slugify(text: string): string {
  return text
    .toLowerCase()
    .replace(/[^\w\s-]/g, '')
    .replace(/\s+/g, '-');
}
```

## Multiple Languages Compared

The same operation in different languages:

```js title="JavaScript"
const response = await fetch('/api/data');
const data = await response.json();
console.log(data);
```

```python title="Python"
import httpx

response = httpx.get('/api/data')
data = response.json()
print(data)
```

```bash title="cURL"
curl -s /api/data | jq .
```
